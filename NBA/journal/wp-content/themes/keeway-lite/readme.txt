== Description ==
Keeway Lite is a clean and beautiful blog theme that is perfect for blogging

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Keeway Lite in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
